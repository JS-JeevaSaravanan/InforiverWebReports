# InforiverWebReports

Serving report JSON data of powerbi reports to Inforiver Standlone and make Inforiver StandAlone functional...




### ReportDataServer

meta.json format

```
{
"report-title":{
         "location":"folder-name/",
         "name": "Report Name",
         "pages": [
             {
                 "label":"Page Lable",
                 "fileName":"file-name.json"
             }
         ]
     }
}
```
